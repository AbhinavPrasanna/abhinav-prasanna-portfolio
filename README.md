# Abhinav Prasanna — Portfolio (React)

A React redesign of [abhinavprasanna.com](https://www.abhinavprasanna.com), built with Vite, TypeScript, and React Router.

## What’s included

- **Home** — Hero, short bio, links to About and Projects  
- **About** — Bio, experience (Wells Fargo, Cognizant, UC Santa Cruz), LinkedIn  
- **Projects** — Main projects (XPM 5, iWare, Home to Home, Minecraft 3js)  
- **Side Projects** — Hometown Water, Bond, Anxiety  
- **Project detail pages** — Full description, tech stack, external links (GitHub, Devpost, etc.)  
- **Responsive layout** — Sticky nav, readable typography, card grid  
- **Theme** — Dark background, teal accent, Outfit + Source Sans 3

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript  
- Vite 5  
- React Router 6  
- CSS Modules (no UI framework)

Content is in `src/data/projects.ts`; edit there to add or change projects.
